# GIT ramas (Branches)

## Crear un repositorio 

```sh
git init
```

## Ver el estado de los archivos

```sh
git status
```

## GIT Alias

```sh
git config --global alias.st "status --short"
git config --global alias.a "add"
git config --global alias.c "commit --m"
git config --global alias.l "log --oneline"
```
